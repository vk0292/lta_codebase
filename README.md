# GDIS Application

GIS Data Management Application built with React, JavaScript, and ArcGIS Maps SDK.

## Features

- Data Management & Upload Functionality
- 3D Model Management
- Borehole Management
- Reporting & Analysis
- Map & Data Synchronization
- Database & Application Migration
- User & Security Management

## Tech Stack

- React 18
- JavaScript (ES6+)
- Vite
- ArcGIS Maps SDK for JavaScript 4.x
- Calcite Design System
- PostgreSQL with PostGIS
- Azure DevOps

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Update .env with your configuration
```

### Development

```bash
# Start development server
npm run dev
```

### Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
